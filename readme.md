## About this Challenge

This ransomware simulation is created purely for educational purposes by the **Reboot Institute Center**.  
It helps you learn cybersecurity skills step-by-step in a safe environment.

---

## Challenges Overview

### Challenge 1 – Malwareinfo  
- **Where:** `/malwareinfo`  
- **What:** Find the secret shutdown password.  
- **Reward:** XP (starts at 1000, decreases over time) + special image.  
- **Goal:** Inspect the page and discover the route — `malwareinfo` and `password.txt`.

---

### Challenge 2 – Picture Steganography  
- **Where:** `/malak`  
- **What:** Find a hidden password inside a picture.  
- **Reward:** +1500 XP  
- **Goal:** Look closely at images — secrets might be hidden inside!

---

### Challenge 3 – Wallet Challenge  
- **Where:** `/check-wallet`  
- **What:** Find the secret wallet password.  
- **Reward:** +1000 XP + a special cookie  
- **Goal:** Discover hidden clues to move forward.

---

### Challenge 4 – Admin Elevation  
- **Where:** `/hello`  
- **What:** Change your cookie to become an admin.  
- **Goal:** Trick the system to unlock the final SQL challenge.

---

### Challenge 5 – IP Magic Sentence  
- **Where:** `/youcantsolveme` (Step 1)  
- **What:** Send the secret magic sentence to the guard:  
  `login from 10.0.0.9`  
- **Reward:** XP + unlock next step to find database tables  
- **Goal:** Discover the special phrase that tricks the guard and reveals more secrets!

---

### Challenge 6 – SQL Injection #1  
- **Where:** `/youcantsolveme` → unlocks `/dbui`  
- **What:** First say a secret phrase, then write a special database query to list all tables.  
- **Reward:** XP (starts at 2000, decreases over time)  
- **Goal:** Learn how to ask the database what it holds.

---

### Final Challenge 7 – Shutdown  
- **Where:** `/shutdown`  
- **What:** Use the final secret password from earlier challenges.  
- **Reward:** Big XP + stop the ransomware timer!  
- **Goal:** Combine what you learned to stop the attack.

---
